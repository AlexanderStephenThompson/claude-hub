# Kelston Chronicles — Domain Language

## World Terms

| Use This | Not This |
|----------|----------|
| Residents | Characters (in Kelston, they *live* here) |
| Regions | Zones, areas, biomes (regions is the canonical term) |
| Locations | Places, landmarks, buildings |
| Lore | History, backstory, canon |
| Anthro | Anthropomorphic, furry (use "anthro") |
| Adapted architecture / resident-adapted | Accessible design, inclusive design (KC's concept is specific: designed for different body types, not just accessibility) |
| Lo-fi | Aesthetic descriptor for site and mood, not a music genre reference |
| Portal | Website, hub (the KC site is framed as a portal into the world) |

## Regions

Each has its own color palette and mood:

| Region | Mood |
|--------|------|
| Tundra | Cool blues, quiet, vast |
| Desert | Warm golds, bright, expansive |
| Rainforest | Deep greens, humid, layered |
| Coastal | Salt-washed blues, breezy |
| Alpine | Crisp whites, elevated |
| Wetlands | Muted greens, still, reflective |
| Urban Canopy | Warm amber, dense, vertical |
| Nocturnal District | Purple/dark, moody, intimate |

## Visual Direction

- **Base:** Dark atmospheric (midnight/charcoal)
- **Accents:** Warm amber lamplight + desaturated blue/teal
- **Surfaces:** Soft gradients, gentle shadows, rounded corners
- **Motion:** Subtle only — slow parallax, drifting dust/stars
- **Per-region:** Color palette shifts when entering a region page
