## Checklist Before Finalizing

- [ ] Reads as a found object, not a wiki entry
- [ ] Written in an in-world voice (not describing Kelston from outside)
- [ ] Leaves at least one question unanswered
- [ ] Has sensory grounding — you can picture where this was found
- [ ] No framing cliches ("it is said that," "legend has it")
- [ ] No urgency words
- [ ] No banned words from `brand/voice.md`
- [ ] Walking pace — short paragraphs, no dense blocks
