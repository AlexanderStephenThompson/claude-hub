# /location-sketch

Write a location description for a place within a Kelston Chronicles region.

## Rules

- Atmospheric and sensory — make the reader feel like they're standing there
- At least one detail per sense used (light, sound, temperature, texture, smell)
- No tourism language — no "bustling," "popular destination," "must-see"
- No explanation of what the place "is" — show what it feels like to be there
- Resident-adapted architecture details where relevant (how the space is designed for different body types)
- Under 100 words — concise and evocative
- Must feel consistent with the region's mood and color palette
- No CTAs, no product language

## Regions and Their Moods

| Region | Mood |
|--------|------|
| Tundra | Cool blues, quiet, vast |
| Desert | Warm golds, bright, expansive |
| Rainforest | Deep greens, humid, layered |
| Coastal | Salt-washed blues, breezy |
| Alpine | Crisp whites, elevated |
| Wetlands | Muted greens, still, reflective |
| Urban Canopy | Warm amber, dense, vertical |
| Nocturnal District | Purple/dark, moody, intimate |
