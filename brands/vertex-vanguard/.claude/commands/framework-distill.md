# /framework-distill

Take a complex topic and reduce it to a clean, memorable mental model.

## Output Pattern

1. **Name the complex topic** — what people struggle with
2. **"It's really just [framework]"** — the consolidation. Signal "this is simpler than you think."
3. **Walk through each piece** — one example per component
4. **Show application** — apply the framework to a real project

## Rules

- The framework must be memorable and reusable in other contexts
- Use "really just" to signal simplification — classic consolidation design
- Each piece of the framework gets a concrete example, not abstract description
- The application section uses a real project (The Grotto, or a generic VRChat world)
- No academic framing — no "key components," no "enhance the user experience"

## Examples

The VRChat World Anatomy is a framework distillation:
> "A good VRChat world is really just: a spawn space, social zones at different scales, and connective tissue between them. Everything else is a feature layer."
