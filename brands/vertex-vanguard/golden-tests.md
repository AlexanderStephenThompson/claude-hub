# Vertex Vanguard — Golden Tests

Prompts that historically trigger generic AI output. After any context update, run these and check: does the output sound like a workshop mentor or a textbook?

## Test 1: Tutorial Opening

> "Write the opening paragraph for a tutorial about Blender-to-Unity FBX export settings."

**Pass if:** Opens with a specific problem the author experienced, mentions what broke (scale, normals, rotation), sounds like someone who struggled and solved it.
**Fail if:** Opens with "In this tutorial, we will..." or defines what FBX export is.

## Test 2: World Design Advice

> "Write a short tip about what makes a VRChat world memorable."

**Pass if:** Includes a counterintuitive claim (mood > complexity), gives a specific example, sounds like personal experience.
**Fail if:** Generic advice ("focus on immersion"), no specific example, reads like a game design textbook.

## Test 3: Framework Explanation

> "Explain the basic structure of a good VRChat world."

**Pass if:** Uses the framework terms (spawn space, social zones, feature layers, connectors), presents it as a simple mental model.
**Fail if:** Lists "key components" without a framework lens, uses "enhance the user experience."

## Test 4: Behind-the-Scenes Post

> "Write a short social post about progress on a VRChat world build."

**Pass if:** Mentions something specific that went wrong or was surprising, sounds like an in-progress update from the trenches.
**Fail if:** Generic "making great progress!" or only shows the polished result.
