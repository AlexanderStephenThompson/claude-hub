# Data Storyteller

Takes raw game data or analysis results and turns them into a narrative-driven data story with the counterintuitive angle, story arc, and visualization suggestions.

## When to Use

- You have raw data (spreadsheet, SQL results, tracking logs) and need a video narrative
- You found something interesting in the data but need to structure the story
- Turning analysis into content — the bridge between data work and video production

## Workflow

1. **Find the angle** — What's the most counterintuitive finding? What common belief does this challenge? That's your hook.
2. **Structure the arc** — Following the Data Story pattern:
   - Hook (counterintuitive stat/claim)
   - Context ("here's what I did")
   - Findings (walk through what the data shows)
   - So what (gameplay implications)
   - CTA (natural, not forced)
3. **Suggest visualizations** — Which data points need charts? What type? (Tableau-style, overlaid on game footage)
4. **Draft the script** — Full script following Data Gamer voice rules
5. **Generate titles and hooks** — 3 title options, 3 hook options

## Output

- Video script (full Data Story format)
- 3 title options
- 3 hook options
- Visualization notes (what to chart, chart type, key callouts)
- Social post for promotion

## Rules

- Gaming-first — the game is the content, data is the proof
- Tools (Tableau, SQL, Excel) are backstage — never lead with them
- Humility about process, confidence about findings
- The story must be interesting even to someone who doesn't care about data
