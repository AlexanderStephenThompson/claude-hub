# /stream-description

Write a Twitch stream description for a Data Gamer stream.

## Rules

- Must mention channel point rewards by name:
  - **Stat Check** — viewer-submitted questions
  - **Challenge the Hypothesis** — viewer-proposed counter-analysis
  - **Show the Spreadsheet** — behind-the-data segment (25K points)
  - **Kill Process** — stream-ending redemption
- Sound like inviting someone to hang out and nerd about game data
- Not educational framing — "come watch me track X" not "learn about data analysis"
- Mention what's being tracked/tested specifically
- Include progression tiers if relevant (Lurker > Regular > Data Nerd > Analyst > Statistician)
