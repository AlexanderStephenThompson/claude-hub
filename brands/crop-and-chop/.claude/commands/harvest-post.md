# Harvest Post

Write a weekly harvest social media post for Crop & Chop.

## Input

The user provides which varieties are in this week's harvest: $ARGUMENTS

## Before Writing

Read these files for voice and pattern guidance:
- `brand/voice.md` -- tone, banned words, testable rules
- `brand/samples.md` -- the "Weekly Harvest Post" named pattern and contrastive examples
- `brand/domain.md` -- correct terminology

## The Weekly Harvest Post Pattern

Follow this structure exactly:

1. **Specific varieties** -- name what's in this week's harvest (use the varieties provided)
2. **Sensory detail about a standout** -- pick one variety and describe its taste, texture, or crunch in a way that makes someone hungry
3. **Subscriber acknowledgment** -- speak directly to subscribers ("yours are packed and ready" or similar)
4. **Market invitation** -- invite everyone else to the market with warmth, not urgency

## Constraints

- Under 50 words (excluding hashtags)
- 1-2 emoji max (zero is fine)
- Lead with flavor, never health claims
- Use "varieties" not "types" or "flavors"
- No banned words (see `brand/voice.md` for full list)
- No urgency language ("don't miss out", "limited", "act now")
- Tone: like a note from a neighbor, not a marketing blast

## Output

One social media post, ready to publish. If it exceeds 50 words, trim it. If it sounds like a marketing department wrote it, rewrite it.
