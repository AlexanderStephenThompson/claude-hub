# Avid Analytics — Domain Language

## Brand Terms

| Use This | Not This |
|----------|----------|
| Clarity / Trust / Leverage | Tier 1/2/3, Basic/Standard/Premium |
| Information architecture | File organization, folder structure (it's more than that) |
| Taxonomy reset | Reorganization, cleanup |
| Data lineage map | Data flow diagram |
| Business glossary | Dictionary, definitions doc |
| Productized services | Consulting packages, service offerings |
| Fixed-scope | Project-based (fixed-scope implies defined deliverables and price, not open-ended) |
| Consolidation design | Systems thinking (Alexander's signature approach) |
| Scope cap | Limit, boundary (scope caps are built into every deliverable to prevent creep) |
| Add-on | Upsell ("add-on" is more honest — priced separately, not hidden) |
| Discovery call | Sales call, consultation |

## Success Metrics

| Tier | Metric |
|------|--------|
| Clarity | "60-second file retrieval" — any file findable in 60 seconds |
| Trust | Documented data lineage, automated pipelines |
| Leverage | AI-ready data, working RAG implementation |

## Positioning Tagline

"Human sanity today, AI readiness tomorrow" — connects Tiers 1-2 to Tier 3.

## Never Say

- Power BI
- IT support
- Help desk
- Troubleshooting
