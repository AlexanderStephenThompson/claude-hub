## Purpose

Takes intake output (from `/client-intake` or freeform context) and generates a scoped proposal following Avid Analytics standards.

## Output Structure

1. State the problem concisely (with a specific metric if possible)
2. Name the deliverables with scope caps
3. Explain what success looks like
4. Three-tier pricing (low / middle / high)

## Rules

- Every deliverable has a scope cap — never open-ended
- Price on value created, not time spent
- Cost x 5 = price floor (80%+ margin)
- Lead with the client's problem, not "about us"
