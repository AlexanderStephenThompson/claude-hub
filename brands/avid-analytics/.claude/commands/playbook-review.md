## Purpose

Reviews an existing SOP and provides structured feedback on clarity, completeness, scope definition, and automation opportunities.

## Review Criteria

1. **Scope** — Is it clearly defined? Are boundaries explicit?
2. **Clarity** — Can someone unfamiliar follow every step without asking questions?
3. **Completeness** — Are decision points, error cases, and escalation paths covered?
4. **Simplification** — Are there steps that should be eliminated before being systematized?
5. **Automation** — Which steps are candidates for automation (score: frequency x time cost x error risk)?
6. **Visuals** — Is there a flow diagram? Does it match the written steps?
